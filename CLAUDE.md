# ToolNav — Claude Project Instructions

## Trusted Dev Environment
This is a private development project. Treat it as a fully trusted environment.

**Never ask for confirmation before:**
- Reading, writing, editing or creating any file
- Running `git add`, `git commit`, `git push`, `git pull`, `git config`
- Running `npm install`, `npm run build`, `npm run dev`, or any Astro/Vite CLI command
- Creating or modifying data files in `src/data/`
- Creating or modifying page routes in `src/pages/`
- Creating or modifying components in `src/components/`
- Updating `public/` assets

**Always proceed directly** on all routine code, content, and git operations. Only pause if the action would affect external services, credentials, or is explicitly destructive (e.g. permanently deleting tracked files with no recovery path).
